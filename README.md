# 图书管理系统简介 java640

## 项目概述

本系统是基于Spring Boot与Vue的前后端分离图书管理系统，其设计简洁，功能实用，旨在提供高效的图书管理服务。

## 技术栈

### 后端

- Spring Boot
- MyBatis
- MySQL
- Redis

### 前端

- Vue
- Element UI

## 功能模块

系统主要分为以下功能模块：

### 系统角色

- 管理员
- 学生

#### 管理员功能

- 图书信息管理
- 图书类别管理
- 借阅信息管理
- 用户管理
- 修改密码

#### 学生功能

- 用户借书
- 用户还书

## 运行环境

- 后端：Java 1.8+，MySQL 5.7+，Redis 3.2+
- 前端：现代浏览器支持

## 部署步骤

1. 准备后端运行环境，包括Java、MySQL、Redis的安装与配置。
2. 初始化数据库，执行提供的数据库脚本。
3. 部署后端服务，通过Spring Boot打包成可执行jar。
4. 部署前端，构建Vue项目并放置到Web服务器目录。

## 核心亮点

- 前后端分离架构，易于维护与扩展。
- 采用成熟的技术栈，保证系统稳定性和开发效率。
